## RSSBlog-Source

本仓库作为[RSSBlog](https://github.com/caibingcheng/rssblog)的资源仓库, 负责rss数据的拉取与整理.

## BRANCH

- master: 代码逻辑
- public: 源数据

数据base url:
```
SOURCE_BASE = "https://raw.githubusercontent.com/caibingcheng/rssblog-source/public/"
```

## FEATURE LIST

- 定时拉取订阅;
- 定时合并订阅;
- 按照时间年月分类;
- 按照用户分类;
- 保留数据来源;
